Panduan Menggunakan Website U KOMPUTER

1. Clone repo ini atau download zip.
2. Pastikan Anda telah menginstal XAMPP, WAMP, atau Laragon untuk menjalankan server Apache dan MySQL. Lalu aktifkan apache dan mysql.
3. Import Database, buka phpMyAdmin, buat database baru, lalu import/copy SQL yang telah disedikan di file sql.sql.
4. Run code saya sarankan dari login.php atau register.php untuk mencoba apakah akun yang dibuat berhasil. NOTE : Untuk mendapatkan previlege ADMIN, buka phpMyAdmin, cari database pwd_akhir, lalu pada tabel users ubah id_role menjadi 1(admin).
5. Setelah login menggunakan akun Admin, pada halaman dashboard kalian bisa mencoba CRUD. Di halaman Produk anda bisa menambah, edit, dan hapus barang.
6. Setelah mencoba CRUD, klik tulisan "U KOMPUTER" pada sidebar untuk masuk ke web utama, untuk melihat barang yg berhasil ditambahkan bisa klik kategori yang ada di navbar atau di konten "Cari Part Komputer".
